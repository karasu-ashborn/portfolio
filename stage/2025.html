Stage 2025 — 2ème année BTS • 2 semaines

Infrastructure Web
& Proxy

Déploiement de services réseau sous Linux : mise en place d'une stack LAMP complète (Apache, PHP, MySQL) et configuration d'un proxy Squid avec règles d'accès et filtrage réseau.

Contexte du stage

Stage de deuxième année de BTS SIO SISR, réalisé sur 2 semaines dans un environnement Linux Debian. Le stage s'est concentré sur deux axes majeurs : le déploiement d'une stack LAMP complète pour héberger des applications web, et la mise en place d'un proxy Squid pour contrôler et filtrer les accès réseau. Ces missions m'ont permis d'approfondir mes compétences en administration Linux, configuration de services web et gestion du flux réseau en entreprise.

Semaine par semaine

Journal de bord

Suivi des 2 semaines de stage et des compétences développées.

Semaine 1

Semaine 1 — Janvier 2025

Déploiement de la stack LAMP

Première semaine consacrée à l'installation et la configuration complète de la stack LAMP sous Linux Debian. Mise en place du serveur web Apache, déploiement de PHP et configuration de la base de données MySQL avec interface phpMyAdmin.

Tâches réalisées

  • Installation et configuration d'Apache (modules, VirtualHosts)
  • Déploiement et configuration de PHP (modules, php.ini)
  • Installation de MySQL / MariaDB et sécurisation initiale
  • Déploiement de phpMyAdmin pour l'administration de la BDD
  • Tests de validation : pages PHP, connexion base de données

Compétences développées

Apache PHP MySQL Administration Linux Services web

Semaine 2

Semaine 2 — Janvier 2025

Proxy Squid & filtrage réseau

Deuxième semaine dédiée à la mise en place du proxy Squid : installation, configuration des ACL et règles d'accès, filtrage réseau, puis validation complète par des tests de connectivité.

Tâches réalisées

  • Installation de Squid sur le serveur Linux
  • Configuration du fichier squid.conf (ports, ACL, règles d'accès)
  • Mise en place de règles de filtrage réseau (whitelist / blacklist)
  • Configuration des postes clients pour passer par le proxy
  • Tests de validation : accès autorisés et URLs bloquées
  • Documentation de la configuration et des règles appliquées

Compétences développées

Proxy Squid Filtrage réseau ACL Configuration réseau Tests & validation

Bilan du stage

Ces 2 semaines m'ont permis de mettre en pratique des compétences concrètes en administration Linux et déploiement de services réseau. Le déploiement de la stack LAMP et la configuration du proxy Squid m'ont donné une maîtrise réelle des services web et de la gestion du flux réseau. Chaque mission a été documentée et validée par des tests, ce qui m'a appris à travailler de façon rigoureuse et méthodique.
